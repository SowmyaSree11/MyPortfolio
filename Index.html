Projects

Automatic Switch using PIR Sensor

This project focuses on automating electrical appliances by detecting human presence using a PIR sensor. The system activates or deactivates the switch based on movement in its detection range.It aims to save energy and enhance convenience by automatically controlling lights and fans. The PIR sensor detects infrared radiation from human bodies, ensuring that appliances are switched on only when needed. This project highlights the application of sensor technology in everyday life for smart automation

KNOW2GROW

KNOW2GROW is an integrated career guidance system designed to assist individuals in making informed career choices. The platform combines personalized learning resources, resume analysis, and career exploration tools to help users navigate their career paths effectively. It provides insightful recommendations based on user profiles, skills, and goals. The system aims to empower users with actionable advice and resources for career advancement. With an intuitive interface, KNOW2GROW aims to bridge the gap between learning and career development.
